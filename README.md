# 🏆 FIFA World Cup 2026 | Media Center

An advanced, high-performance web platform designed for low-latency technical media distribution, global broadcasting feeds, and real-time match schedule tracking for the FIFA World Cup 2026.

## ✨ Features & Advantages

* **Multi-Source Network Architecture** Features an integrated sidebar hosting premium global broadcast feeds, seamlessly aggregating official FIFA+ streams (UK, US, and Mediacorp Singapore) alongside industry-leading sports networks like Fox Sports, FOX Deportes, and CazéTV Brazil.

* **Advanced Multi-Format Video Player** Built on top of a highly responsive Video.js engine capable of natively decoding adaptive bitrate protocols, including HTTP Live Streaming (`.m3u8` HLS) and Dynamic Adaptive Streaming over HTTP (`.mpd` DASH) up to Full HD 1080p.

* **CORS Proxy Integration** Engineered with network-level proxy wrappers to bypass Cross-Origin Resource Sharing (CORS) restrictions, preventing player loading failures and ensuring stable stream delivery directly inside production web environments.

* **Dynamic Internationalization (i18n)** Includes a built-in, client-side language localization toggle supporting seamless switching between English (EN) and Malay (MY) across all UI components and live match metadata.

* **Automated System Time Conversion** Equipped with an intelligent JavaScript schedule engine that ingests standardized ISO UTC (`Z`) match vectors and instantly translates them into the end-user's local device/system time zone (e.g., Malaysia Time GMT+8).

* **Premium Glassmorphism Interface** Boasts a cutting-edge, cyber-inspired dark mode layout styled with high-fidelity blur backdrops, responsive grid layouts, status pulse telemetry indicators, and custom smooth-scrolling containers.
